# shadcn/ui monorepo template

This project follows the setup from `Phases/02-setup.md`: a Turborepo workspace with two Next.js apps, shared UI, and an internal math package.

## Apps

- `apps/web` runs on port `3000`
- `apps/widget` runs on port `3001`

## Commands

```bash
pnpm install
pnpm dev
pnpm build
```
